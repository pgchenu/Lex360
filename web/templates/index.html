Configurer le token JWT

Connectez-vous sur lexis360intelligence.fr, puis récupérez le token depuis la console du navigateur :

Chrome
  1. Appuyez sur F12 ou Cmd+Option+J
  2. Allez dans l'onglet Console
  3. Tapez : localStorage.getItem('access_token')
  4. Copiez la valeur retournée (sans les guillemets)
Firefox
  1. Appuyez sur F12 ou Cmd+Option+K
  2. Allez dans l'onglet Console
  3. Tapez : localStorage.getItem('access_token')
  4. Copiez la valeur retournée (sans les guillemets)

Le token expire au bout de 24h. Renouvelez-le en vous reconnectant sur Lexis 360.
